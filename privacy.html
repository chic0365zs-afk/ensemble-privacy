앙상블(Ensemble) 개인정보처리방침

시행일: 2026년 4월 28일

앙상블(Ensemble, 이하 "서비스")은 「개인정보 보호법」 및 관련 법령을 준수하며, 이용자의 개인정보 보호를 위해 다음과 같은 처리방침을 두고 있습니다.

1. 수집하는 개인정보 항목

앙상블(Ensemble)은 서비스 제공을 위해 다음과 같은 개인정보를 수집합니다.

필수 항목

  • 이메일 주소, 닉네임, 프로필 사진(소셜 로그인 제공)
  • Firebase 계정 식별자(UID)

선택 항목

  • 지역(시/도, 구/군), 생년월일, 성별

서비스 이용 시 자동 수집

  • 위치 정보(GPS 좌표): 활동 기록 및 빛 릴레이 기능
  • 기기 정보: FCM 푸시 토큰, 광고 식별자(AAID/IDFA)
  • 서비스 이용 기록: 접속 일시, 활동 통계, 앱 충돌 정보(Crashlytics)

활동 기록

  • 환경 활동 사진, GPS 위치, 활동 시간, 활동 카테고리, 사용자 입력 메모

사용자 입력 데이터

  • 커뮤니티 게시글, 댓글, 신고 내역, 차단 목록
  • 다이어리 텍스트(AI 분석 활용)

민감정보(별도 동의 시에만 처리)

  • 건강 정보: Health Connect 걸음수(보상 지급 목적)
※ 「개인정보보호법」 제23조에 따라 별도 동의를 받아 처리하며, 거부 시에도 다른 서비스 이용에는 제한이 없습니다.

비즈니스 파트너 신청 시

  • 매장명, 매장 주소, 사업자 정보, 제품 사진

AI 자동 생성 데이터

  • 사용자 활동 패턴 분석 결과(identity, chapters, insights, proposals)
※ 서비스 이용에 필수적인 항목 외에는 동의를 거부할 수 있으며, 거부 시 일부 기능이 제한될 수 있습니다.

2. 수집 및 이용 목적

  • 회원 가입 및 본인 확인: 이메일, 닉네임, UID
  • 환경 활동 인증: 사진, 위치, 시간, AI 검증
  • 나무 성장 시스템: 활동 통계, 보상 지급
  • 빛 릴레이(위치 기반): 근거리 유저 탐색
  • 푸시 알림: 활동 리마인더, 릴레이 알림
  • 커뮤니티 운영: 게시글/댓글 표시, 신고 처리
  • AI 다이어리: 활동 패턴 분석으로 맞춤 인사이트 제공
  • 환경 정보 제공: 사용자 지역의 대기질·날씨 안내
  • 광고 게재: AdMob 리워드 광고
  • 실제 나무 심기: Digital Humani API 연동
  • 서비스 개선 및 통계 분석: 앱 안정성, 충돌 보고

3. 보유 및 이용 기간

  • 회원 탈퇴 시까지 보유 (탈퇴 시 즉시 파기)
  • 단, 관련 법령(전자상거래법, 통신비밀보호법 등)에 따라 일정 기간 보존이 필요 시 해당 기간 동안 보관
  • 위치 정보: 마지막 활동일로부터 1년
  • 활동 사진: 회원 탈퇴 시 즉시 삭제
  • 앱 충돌 보고(Crashlytics): 90일 (Google 정책)
  • 결제 기록(향후 인앱결제 시): 5년 (전자상거래법)

4. 제3자 제공

앙상블은 원칙적으로 이용자의 개인정보를 외부에 제공하지 않습니다. 다음의 경우는 예외입니다.

  • 이용자가 사전에 동의한 경우
  • 법령의 규정에 의거하거나 수사 목적으로 법령에 정해진 절차와 방법에 따라 수사기관의 요구가 있는 경우
  • 통계 작성, 학술 연구 등의 목적을 위해 특정 개인을 식별할 수 없는 형태로 제공하는 경우

5. 처리 위탁

앙상블은 원활한 서비스 제공을 위해 다음과 같이 개인정보 처리 업무를 위탁합니다.

Google LLC

  • Firebase Authentication: 계정 인증
  • Firebase Firestore: 사용자 데이터 저장
  • Firebase Storage: 활동 사진 저장
  • Firebase Cloud Messaging(FCM): 푸시 알림 발송
  • Firebase Crashlytics: 앱 충돌 분석
  • Firebase App Check: 비정상 접근 차단
  • Cloud Functions: 서버 로직 실행
  • Google Gemini API: 활동 사진 AI 검증, AI 다이어리 생성
  • Google Cloud Vision API: 사진 SafeSearch (부적절 이미지 필터)
  • Google AdMob: 리워드 광고 게재 (광고 식별자 수집)
  • Google Maps API: 지도 기능

Mapbox, Inc. (미국)

  • Mapbox API: 3D 지도, 글로브 시각화

Kakao Corp.

  • Kakao Local API: 위치 좌표를 주소로 변환

한국환경공단, 기상청

  • 공공데이터포털(data.go.kr): 대기질·날씨 정보 조회

Digital Humani (캐나다)

  • 나무 심기 API: 사용자 ID 및 프로젝트 선택 정보 전달

수탁업체는 위탁받은 업무 목적 외 개인정보를 처리하지 못하도록 계약 및 정책을 통해 관리·감독하고 있습니다.

서비스 제공 과정에서 다음과 같이 개인정보가 국외로 이전될 수 있습니다.

Google LLC (미국)

  • 이전 항목: 위 "5. 처리 위탁" 항목과 동일
  • 이전 국가: 미국 (데이터 센터)
  • 이전 일시 및 방법: 서비스 이용 시 네트워크를 통한 자동 전송
  • 보유 기간: 회원 탈퇴 시까지

Mapbox, Inc. (미국)

  • 이전 항목: IP 주소, 지도 요청 정보
  • 이전 국가: 미국
  • 이전 방법: 지도 화면 표시 시 자동 전송

Digital Humani (캐나다)

  • 이전 항목: 사용자 ID, 프로젝트 선택 정보
  • 이전 국가: 캐나다
  • 이전 방법: 나무 심기 인증 시 자동 전송

이용자는 국외 이전을 거부할 수 있으나, 거부 시 해당 기능 이용이 제한됩니다.

7. 파기 절차 및 방법

회원 탈퇴 시 개인정보는 다음과 같이 파기됩니다.

  • Firestore 문서: 사용자 문서 및 하위 컬렉션(활동, 다이어리, 게시글) 즉시 삭제
  • Firebase Storage: 업로드된 활동 사진, 프로필 사진 삭제
  • Firebase Authentication: 계정 정보 영구 삭제
  • 위탁사 데이터: 각 수탁업체 정책에 따라 순차 삭제
  • 법령에 따라 보관이 필요한 정보: 별도 분리 보관 후 기간 만료 시 파기

전자적 파일은 복구 불가능한 방법으로 영구 삭제됩니다.

8. 만 14세 미만 아동의 개인정보

앙상블은 만 14세 미만 아동의 개인정보를 수집하지 않습니다. 만 14세 미만이 서비스를 이용하려면 법정대리인의 동의가 필요합니다.

회원 가입 시 생년월일 입력을 통해 만 14세 미만 여부를 확인하며, 해당 사실이 확인될 경우 즉시 해당 정보를 삭제하고 회원 자격을 제한합니다.

10. 위치 정보 처리

앙상블은 「위치정보의 보호 및 이용 등에 관한 법률」에 따라 위치 정보를 처리합니다.

  • 수집 항목: GPS 좌표(위도, 경도)
  • 수집 시점: 환경 활동 제출 시, 빛 릴레이 매칭 시
  • 이용 목적: 활동 위치 기록, 근거리 유저 탐색
  • 보관 기간: 마지막 활동일로부터 1년
  • 동의 거부: 단말기 위치 권한 거부 시 활동 제출 및 빛 릴레이 기능 이용 제한
  • 사업자 신고: 「위치정보법」 제9조에 따라 위치기반서비스 사업자 신고를 진행 중입니다.

11. 광고 식별자(AdMob)

앙상블은 Google AdMob을 통해 리워드 광고를 게재하며, 광고 표시 및 측정을 위해 광고 식별자를 수집합니다.

  • Android: AAID (Google Advertising ID)
  • iOS: IDFA (Identifier for Advertisers)
  • 이용 목적: 광고 표시, 사기 방지, 광고 효과 측정

광고 식별자 거부 방법

  • Android: 설정 > Google > 광고 > 광고 ID 재설정/삭제
  • iOS: 설정 > 개인 정보 보호 > 추적 > 앱이 추적 요청 허용 끄기

12. 광고성 정보 수신

앙상블은 다음과 같이 푸시 알림을 발송합니다.

서비스 알림 (필수, 동의 불요)

  • 빛 릴레이 매칭, 활동 리마인더, 보상 지급 안내

광고성 정보 (별도 동의 후 발송)

  • 이벤트, 프로모션, 신규 기능 안내

광고성 정보 수신 거부

  • 앱 내: 설정 > 알림 설정에서 광고성 알림 OFF
  • 단말 설정: 시스템 알림 권한 차단

13. 안전성 확보 조치

앙상블은 개인정보의 안전성 확보를 위해 다음과 같은 조치를 취하고 있습니다.

  • 전송 구간 암호화: TLS 1.2 이상
  • Firebase App Check: 정상 앱에서만 데이터 접근 허용
  • 접근 권한 제어: Firestore 보안 규칙으로 본인 정보만 접근
  • 비밀번호 미저장: 소셜 로그인 사용 (Google)
  • 정기 보안 점검 실시

유출 시 조치

「개인정보보호법」 제34조에 따라 개인정보 유출 발생 시 72시간 이내에 정보주체에게 유출 항목, 시점, 대응 조치 등을 통지합니다.
